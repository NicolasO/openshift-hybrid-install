# openshift-hybrid-install

This repo is used for the installation of the OCP cluster based on OVA template from Red Hat for Master+Infra nodes and HPE Synergy BareMetal Servers for Workers.



## Pre-requisites

- DNS Server
- DHCP Server
- NTP Server
- HAProxy Server
